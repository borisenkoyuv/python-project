## Начальная страница sign in

### Позитивные кейсы

### TC-SIGN-IN-001 Проверка смены языка

**Предусловия:** Страница sign in открыта 
**Шаги:**
1. Кликнуть на меню выбора языка   
2. Выбрать язык, отличный от текущего

**Ожидаемый результат:** 
1. Текстовые элементы на странице соответствуют выбранному языку

### TC-SIGN-IN-002 Проверка открытия ссылок

**Предусловия:** Страница sign in открыта 
**Шаги:**
1. Кликнуть по ссылке (help, privacy, terms, learn more about using guest mode)

**Ожидаемый результат:** 
1. В новой вкладке открывается корректная ссылка

### TC-SIGN-IN-003 Проверка перехода по ссылке восстановления пароля

**Предусловия:** Страница sign in открыта 
**Шаги:**
1. Кликнуть по ссылке (help, privacy, terms, learn more about using guest mode)

**Ожидаемый результат:** 
1. В новой вкладке открывается корректная ссылка

### TC-SIGN-IN-004 Проверка кнопки создания аккаунта

**Предусловия:** Страница sign in открыта 
**Шаги:**
1. Кликнуть по кнопке создания аккаунта
2. Выбрать один из разделов (For my personal use, For my child, For work or my business)

**Ожидаемый результат:** 
1. В новой вкладке открывается форма создания аккаунта с текстом, соответствующим выбранному разделу

### TC-SIGN-IN-005 Проверка распознавания email по имени пользователя

**Предусловия:** Страница sign in открыта 
**Шаги:**
1. Ввести в поле email только имя пользователя
2. Кликнуть по кнопке Далее

**Ожидаемый результат:** 
1. На следующей странице открывается отображается полноценный почтовый адрес

### Негативные кейсы

### TC-SIGN-IN-006 Проверка ввода пустого email/телефона

**Предусловия:** Страница sign in открыта 
**Шаги:**
1. Ввести пустое значение в поле email
2. Кликнуть на кнопку Далее

**Ожидаемый результат:** 
1. Перехода на следующую страницу не происходит
2. Появляется сообщение об ошибке
